# SmartHire AI Agent

A free local AI resume screening agent built with Streamlit and Ollama.

## Features
- Upload resume PDF
- Paste job description
- Extract resume text
- Compare resume with job description
- Generate match score
- Shortlist / Human Review / Reject decision
- No paid API key required

## Tech Stack
- Python
- Streamlit
- pdfplumber
- Ollama
- Llama 3.2 local model

## Run Locally

Install dependencies:

```bash
pip install -r requirements.txt
